# Readify — Full-Stack E-Commerce Web App

Readify is a production-ready online bookstore built using ASP.NET Core MVC and Clean Architecture principles.  
The project simulates a real-world e-commerce platform with secure authentication, multi-payment integration, admin dashboard management, and scalable architecture.

# Features

## Customer Features

- Browse & filter 128 books across 12 categories
- Product details with dynamic pricing tiers
- Full shopping cart & checkout workflow
- Order tracking & order history
- Secure authentication & authorization
- Email confirmation & password reset
- Google OAuth login
- Multi-payment gateway support
- Responsive modern UI

## Payment Integrations

- Stripe Integration (International Payments)
- Kashier Integration (Egyptian Payments)
- Refund Support
- Extensible Multi-Gateway Payment Architecture

## Admin Dashboard

- Products CRUD
- Categories CRUD
- Image Upload Management
- Orders Management
- Order Status Updates
- Users & Roles Management
- Dashboard Analytics

## Email Services

- Email Confirmation
- Password Reset Emails
- Order Notifications
- Twilio SendGrid Integration
